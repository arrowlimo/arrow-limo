⚙️ .ENV CONFIGURATION (Y:\ArrowLimo\app\.env):

# Add these lines to .env if not present:

# Database timeout protection (prevents hangs)
DB_CONNECT_TIMEOUT=30          # Max 30 seconds to establish connection
DB_STATEMENT_TIMEOUT=300       # Max 5 minutes per query (300 seconds)
DB_KEEPALIVES=0                # Disabled for localhost, set to 1 for cloud

# Optional keepalive settings (only needed if DB_KEEPALIVES=1):
DB_KEEPALIVES_IDLE=30          # Start keepalive after 30s idle
DB_KEEPALIVES_INTERVAL=10      # Probe every 10s
DB_KEEPALIVES_COUNT=5          # 5 failed probes = dead connection

Issue: "Connection still takes forever to timeout"
  → Check: Did you restart the application after deployment?
  → Check: Is .env file in correct location? (Y:\ArrowLimo\app\.env)
  → Check: Run test script to verify configuration

  → Fix: Increase DB_STATEMENT_TIMEOUT in .env (e.g., 600 for 10 minutes)
  → Restart application
