🔍 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

**Problem: Always says "OFF-SITE" even when in office**
Solution:
  • Check DISPATCHMAIN_IP in smart_launcher.py
  • Make sure DISPATCHMAIN is turned on
  • Verify you're on same network as DISPATCHMAIN
  • Test: ping [DISPATCHMAIN_IP]

**Problem: Cannot connect to either database**
Solution:
  • Check internet connection (needed for cloud)
  • Verify Neon cloud credentials in .env.cloud
  • Make sure PostgreSQL running on DISPATCHMAIN (for local)

**Problem: Using wrong database**
Solution:
  • Check console output when launching
  • Look for "IN OFFICE" or "OFF-SITE" message
  • Verify .env file contents (DB_HOST shows which is active)

**Problem: Want to force a specific database**
Solution:
  • Copy .env.local to .env (force local)
  • OR copy .env.cloud to .env (force cloud)
  • Skip smart launcher, use regular launcher
