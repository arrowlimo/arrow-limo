DISPATCHMAIN Scripts (Run on DISPATCHMAIN first):
──────────────────────────────────────────────────

📜 GET_DISPATCHMAIN_IP.ps1
   Purpose: Display DISPATCHMAIN IP address for DISPATCH1 setup
   Usage: .\GET_DISPATCHMAIN_IP.ps1
   Output: IP address, network info, PostgreSQL status

📜 SETUP_POSTGRESQL_NETWORK_ACCESS.ps1
   Purpose: Configure PostgreSQL for network connections
   What it does:
     - Modifies postgresql.conf (listen_addresses)
     - Updates pg_hba.conf (allow local network)
     - Opens firewall port 5432
     - Restarts PostgreSQL service
     - Creates config backups
   Usage: .\SETUP_POSTGRESQL_NETWORK_ACCESS.ps1
   Note: Run as Administrator

📜 SETUP_NETWORK_SHARE_FOR_AUTOUPDATE.ps1
   Purpose: Create network shares for L: and Z: drives
   Creates: \\DISPATCHMAIN\limo and \\DISPATCHMAIN\limo_files
   Usage: .\SETUP_NETWORK_SHARE_FOR_AUTOUPDATE.ps1

DISPATCH1 Scripts (Run on DISPATCH1):
─────────────────────────────────────

📜 TEST_DATABASE_CONNECTION.ps1
   Purpose: Test connectivity from DISPATCH1 to DISPATCHMAIN database
   What it tests:
     - Ping DISPATCHMAIN
     - Port 5432 accessibility
     - PostgreSQL login
     - Database stats
   Usage: .\TEST_DATABASE_CONNECTION.ps1

📜 DispatchInstaller_Dispatch1.ps1
   Purpose: Install application on DISPATCH1 (older version)

📜 INSTALL_DISPATCH1_Y_DRIVE.bat

🆘 COMMON QUESTIONS
═══════════════════════════════════════════════════════════════════════════

Q: Which script should I use on DISPATCH1?
A: Use SETUP_DISPATCH1_COMPLETE.ps1 - it does everything automatically.

Q: Do I need all these files?
A: Technically no, but keep them all for troubleshooting and documentation.

Q: What's the difference between the installers?
A: SETUP_DISPATCH1_COMPLETE.ps1 is newest and most comprehensive.
   DispatchInstaller_Dispatch1.ps1 is older, manual steps.
   COMPLETE_SETUP.bat is legacy (C: drive installation).

Q: Can I run this without VS Code?
A: Yes! Use PowerShell directly. See QUICK_START.md.

Q: What if automation fails?
A: Follow manual steps in QUICK_START.md or NETWORK_DATABASE_SETUP_GUIDE.md.

Q: How do I uninstall?
A: See README.txt "UNINSTALLING" section or DISPATCH1_VS_CODE_SETUP.txt.

Q: Where are error logs?
A: Y:\ArrowLimo\app_errors.log (DISPATCH1)
   L:\limo\app_errors.log (DISPATCHMAIN)

Q: Can I reinstall?
A: Yes! Run SETUP_DISPATCH1_COMPLETE.ps1 with -ForceReinstall flag.

Q: Do I need internet?
A: Only for Python package installation (pip install).
   Database works 100% offline on local network.

Q: What Python version?
A: Python 3.10 or higher required.
