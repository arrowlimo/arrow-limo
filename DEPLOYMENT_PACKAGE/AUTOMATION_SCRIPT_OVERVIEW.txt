TROUBLESHOOTING THE AUTOMATION
═══════════════════════════════════════════════════════════════════════════

Script won't run:
→ Set execution policy: Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
→ Verify running as Administrator
→ Check script not blocked: Unblock-File .\SETUP_DISPATCH1_COMPLETE.ps1

Y: drive creation fails:
→ Close all programs using Y:
→ Unmount if exists: Dismount-VHD -Path "$env:USERPROFILE\ArrowLimoY.vhdx"
→ Delete old VHDX: Remove-Item "$env:USERPROFILE\ArrowLimoY.vhdx" -Force
→ Retry script

Network drive mapping fails:
→ Verify \\DISPATCHMAIN\limo and \\DISPATCHMAIN\limo_files shares exist
→ Check network credentials
→ Manually test: net use L: \\DISPATCHMAIN\limo
→ Check DISPATCHMAIN sharing is enabled

Database connection fails:
→ Verify DISPATCHMAIN IP address
→ Check PostgreSQL password
→ Run on DISPATCHMAIN: .\SETUP_POSTGRESQL_NETWORK_ACCESS.ps1
→ Test port: Test-NetConnection -ComputerName [IP] -Port 5432

Python packages fail to install:
→ Update pip: python -m pip install --upgrade pip
→ Check internet connection (for package download)
→ Try manual install: pip install psycopg2-binary PyQt6 python-dotenv
→ Verify Python path: python --version
