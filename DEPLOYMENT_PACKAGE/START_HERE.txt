🎓 KEY CONCEPTS TO UNDERSTAND
═══════════════════════════════════════════════════════════════════════════

**1. DUAL DISPATCHERS = Real-Time Shared Database**
   • Both DISPATCHMAIN and DISPATCH1 connect to same PostgreSQL
   • Changes appear instantly on both machines
   • No conflicts (PostgreSQL handles multi-user)
   • Database stays on C: drive (DISPATCHMAIN)

**2. HYBRID = Automatic Local/Cloud Switching**
   • Smart launcher detects location
   • In office: Uses local (fast!)
   • Off-site: Uses cloud (accessible)
   • No manual configuration needed

**3. TWO DATABASES = Must Manually Sync**
   • Local PostgreSQL and Neon cloud are separate
   • Changes to one don't affect the other
   • Use sync utility to keep aligned
   • Choose primary database (recommend local)

**4. AUTO-UPDATE = Code Syncs Automatically**
   • DISPATCH1 syncs from L:\limo on launch
   • Edit code on DISPATCHMAIN
   • DISPATCH1 gets updates automatically
   • Restart DISPATCH1 app to apply changes
