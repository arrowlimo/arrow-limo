## SEARCH INSTRUCTIONS

### In Outlook:

1. **Open Outlook PST** (Heffner/CMB folder)
2. **Use Find & Replace** (Ctrl+H or Edit menu)
3. **Search for:** Reserve number (e.g., "019843")
4. **Look for:**
   - Email subjects with reserve number
   - Attachments (receipts, invoices, confirmations)
   - Body text mentioning amount
   - Any transaction reference

### Search Tips:
- Try "019843" and "19843" (with and without leading zero)
- Search in Subject line + Message Body
- Check sent items (outgoing confirmations)
- Check Calendar (if payments tracked in calendar notes)
- Look for check numbers or deposit confirmation codes

### When You Find a Match:
Record the following:
```
Reserve: 019843
Payment Amount Found: [AMOUNT]
Payment Date Found: [DATE]
Source: [EMAIL DATE / ATTACHMENT TYPE]
Notes: [Any relevant details]
Status: ✅ FOUND
```

### When You Don't Find a Match:
Record:
```
Reserve: 019843
Searched: [AREAS SEARCHED: Subject, Body, Attachments, etc.]
Status: ❌ NOT FOUND
Notes: [Any relevant details, e.g., "Searched all folders 2025-2026"]
```
